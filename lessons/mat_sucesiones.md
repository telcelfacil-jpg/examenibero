# Sucesiones Aritméticas y Geométricas

Una sucesión es un conjunto de números que siguen un orden o patrón específico.

## 1. Sucesión Aritmética

Es aquella donde cada término se obtiene **sumando o restando** una cantidad fija (llamada diferencia) al término anterior.

**Ejemplo:** 2, 5, 8, 11...

- El patrón es **+3**.
- Siguiente término: 11 + 3 = 14.

## 2. Sucesión Geométrica

Es aquella donde cada término se obtiene **multiplicando** por una cantidad fija (llamada razón) al término anterior.

**Ejemplo:** 3, 6, 12, 24...

- El patrón es **x2**.
- Siguiente término: 24 x 2 = 48.

> [!TIP]
> **En el examen:** Si los números crecen muy rápido, busca una multiplicación. Si crecen poco a poco, busca una suma.
