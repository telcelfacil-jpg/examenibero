# Media, Mediana y Moda

Estas medidas nos ayudan a resumir un conjunto de datos en un solo valor central.

## 1. Media (Promedio)

Suma todos los valores y divide el resultado entre la cantidad de datos.

- Ejemplo: (5 + 7 + 9) / 3 = 7.

## 2. Mediana

Es el valor que ocupa el **lugar central** cuando los datos están ordenados de menor a mayor.

- Si hay un número par de datos, es el promedio de los dos centrales.

## 3. Moda

Es el valor que **más veces se repite** en el conjunto de datos.

> [!IMPORTANT]
> **Cuidado**: Si ningún número se repite, no hay moda. Si hay dos que se repiten por igual, es bimodal.
